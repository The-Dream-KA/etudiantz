# EtudiantZ Platform Context

## Overview

**EtudiantZ** is a "SUPER APP" designed to serve both **Students** and **Employers** in Belgium. The platform is built with a multilingual-first approach, supporting French (FR), Dutch (NL), and English (EN).

## Core User Flows

The application is split into two main user flows:

1.  **Student Flow:** Tailored services and features for students.
2.  **Employer Flow:** Dedicated tools and functionalities for employers.
